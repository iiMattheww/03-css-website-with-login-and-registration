# Website — Login & Registration UI

A small HTML/CSS/JS demo implementing a login + registration modal UI (inspired by Codehal).

Features:
- Pure HTML/CSS UI with minimal JavaScript for modal and tab toggles.
- Uses Ionicons for icons; no backend — frontend UI only.

Quick start:
- Open `index.html` in your browser.
- Or serve locally (PowerShell):
	- `python -m http.server 8000`
	- Visit `http://localhost:8000`
	- (Or) `npx http-server -p 8000`

Files:
- `index.html` — markup
- `style.css` — styling
- `script.js` — modal toggle logic
- `LICENSE` — project license
